Privacy Policy

1. What data do we collect?

We collect and store basic Discord identifiers including User IDs, Server IDs, and Channel IDs. This data is securely written to localized json files to preserve configuration settings through bot reboots.

Additionally, transient message data is cached in temporary memory for a maximum duration of two hours before being permanently dropped.

2. How do we use this data?

  • Identifiers (IDs): Utilized to keep track of global economy balances, level statistics, server leaderboards, board features, and to restrict access to guilds violating our Terms of Service.
  • Cached Messages: Utilized strictly to power automated features like ghost-ping detection alerts, board updates, and the /deleted or /edited utilities.

3. Who do we share data with?

Absolutely nobody. Data access is exclusively limited to the primary developer (ImNinnn). Under normal operating conditions, live message content cannot be manually monitored or viewed.

4. How do I request data removal?

If you want all records pertaining to your profile deleted, simply join our official support server and drop a message in our designated data-removal channel.
